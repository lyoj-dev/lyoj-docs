# 获取评测状态 (不含各测试点状态)

```bash
GET /api/status/status.php
```

**参数**

| 变量名 | 类型 | 必要性 |  含义   | 备注 |
| :----: | :--: | :----: | :-----: | :--: |
|  `id`  | num  |  必要  | 评测 id |  -   |

**返回值**

|   字段   |  类型  |   内容   | 备注 |
| :------: | :----: | :------: | :--: |
|   `id`   |  num   | 评测 id  |  -   |
| `result` | string | 评测状态 |  -   |
