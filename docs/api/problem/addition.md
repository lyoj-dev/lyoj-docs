# 下载附加文件

```bash
GET /api/problem/addition.php
```

**参数**

| 变量名 |  类型  | 必要性 |     含义     | 备注 |
| :----: | :----: | :----: | :----------: | :--: |
| `pid`  |  num   |  必要  |    题目id    |  -   |
| `name` | string |  必要  | 附加文件名称 |  -   |

**返回值**

本 API 不返回 JSON，直接返回下载头

-----

#### 提交代码

```bash
POST /api/problem/submit.php
```

**参数**

| 变量名 |  类型  | 必要性 |  含义  |               备注               |
| :----: | :----: | :----: | :----: | :------------------------------: |
| `pid`  |  num   |  必要  | 题目id |                -                 |
| `code` | string |  必要  |  代码  |                -                 |
| `lang` |  num   |  必要  | 语言id | 具体见 [语言代码选项](../others/lang) |

**返回值**

|  字段  |  类型  |   内容   | 备注 |
| :----: | :----: | :------: | :--: |
|  `id`  |  num   | 评测 id  |  -   |
| `lang` |  num   | 提交语言 |  -   |
| `code` | string | 提交代码 |  -   |
