# 重传题目数据

```bash
POST /api/problem/upload.php
```

**参数**

| 变量名 |  类型  | 必要性 |     含义      | 备注 |
| :----: | :----: | :----: | :-----------: | :--: |
| `file` | string |  必要  | 数据包 base64 |  -   |
| `pid`  |  num   |  必要  |    题目 id    |  -   |

**返回值**

| 字段 | 类型 |  内容   | 备注 |
| :--: | :--: | :-----: | :--: |
| `id` | num  | 题目 id |  -   |
