# SHA256 加密

```bash
POST /api/tool/crypt.php
```

**参数**

| 变量名 |  类型  | 必要性 |     含义     | 备注 |
| :----: | :----: | :----: | :----------: | :--: |
| `data` | string |  必要  | 要加密的数据 |  -   |
| `key`  | string |  必要  |     公钥     |  -   |

**返回值**

|   字段   |  类型  |   内容   | 备注 |
| :------: | :----: | :------: | :--: |
| `result` | string | 加密结果 |  -   |
